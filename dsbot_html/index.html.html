Discord bot

Описание

Этот бот создавался для помощи людям у которых есть сервер в дискорде, которые хотели разнообразить быт в дискорде.


Этот бот может:

  1. Когда на сервер заходит челоке он выдает ему роль.
  2. Когда человек выходи с сервера , бот забирает роль.
  3. С помощью команды /hb , может определённого человека поздравить с днём рождения.
  4. Также с помощью команды , админы сервера могут отправить определённого человека отправить в тайм-аут.
  5. По команде /guess_number , если угадать число , то бот выдаст секретную роль.
